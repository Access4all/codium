Alles mit ARIA bitte!

Auf dieser Seite werden relevante Beispiele zur Verwendung verschiedener ARIA-Attribute aufgeführt.

arialabelledby=""

Hier ist eine Liste von Links umgesetzt. Der Link "Peter" besitzt ein aria-labelledby attribut und sollte auf die Überschrift dieses Abschnittes verweisen. Ein Screenreader sollte somit beim Fokussieren des Links "Peter" den Text der Überschrift erneut ausgeben (aria-labelledby="")

mögliche Stolpersteine

Ist es möglich, dass das labelledby Element nicht vom Screenreader ausgegeben wird, wenn sich innerhalb des verlinkten Elementes weitere Elemente mit eigenem Inhalt befinden?

Der Link "Peter" weist erneut ein aria-labelledby Attribut auf, doch diesmal ist das Ul-Element der vorherigen Liste verlinkt. Dieses Ul-Element beinhaltet mehrere Elemente, die wiederum eigenen Text beinhalten. Was wird nun ausgegeben, wenn der Screenreader den Link "Peter" fokussiert?
